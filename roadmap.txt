 What You Need to Do:

  1. Get Your API Keys

  - NVIDIA API Key: Go to https://build.nvidia.com, sign up, create an API key
  - Supabase:
    - Create project at https://supabase.com
    - Run supabase-setup.sql in SQL Editor
    - Get URL and Service Role key from Settings → API

  2. Edit .dev.vars

  Open .dev.vars and replace:
  NVIDIA_API_KEY=sk-your-nvidia-api-key-here
  SUPABASE_URL=https://your-project.supabase.co
  SUPABASE_KEY=your-supabase-service-role-key-here

  3. Run Setup & Start Dev Server

  Once you've filled in your keys:
  pnpm setup
  wrangler dev

  Or all at once:
  pnpm setup && wrangler dev

  4. Open Your Browser

  Navigate to: http://127.0.0.1:8787
  
  
  
  npx wrangler dev --port 8787
